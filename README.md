## Über das Projekt

Dieses Repository enthält praxisnahe Anleitungen und Code-Beispiele für Entwickler und Institutionen, die XRPL mit traditioneller Finanzinfrastruktur (DTCC, Ripple Prime) kombinieren wollen.

### Wichtige Themen
- Getting Started mit Ripple Prime (NSCC 0443)
- Collateral Optimization & Rebalancing
- Chainlink CCIP Integration
- Tokenized Assets & RLUSD als Collateral
